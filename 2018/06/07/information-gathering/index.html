信息搜集

发表于 2018-06-07 | 更新于 2020-03-07 | 分类于 pentest |

信息搜集通常要搜集以下几类信息：

  • DNS 各类型解析信息
  • 域名 WHOIS，备案情况
  • 子域名列表
  • 操作系统
  • 端口及服务
  • 社工信息（人员，邮箱，电话，历史泄露密码）
  • 源码泄露.svn/.git 文件（github 及主要的公共仓库）
  • C 段
  • 路径及服务
  • 网络空间资产
